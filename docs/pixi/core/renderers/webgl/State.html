package pixi.core.renderers.webgl

@:native("PIXI.State")

Available on all platforms

Constructor

new()

This is a WebGL state, and is is passed The WebGL StateManager. Each mesh rendered may require WebGL to be in a different state. For example you may want different blend mode or to enable polygon offsets

Variables

blend:Bool

Activates blending of the computed fragment color values

blendMode:Int

The blend mode to be applied when this state is set. Apply a value of PIXI.BLEND_MODES.NORMAL to reset the blend mode. Setting this mode to anything other than NO_BLEND will automatically switch blending on.Â´

clockwiseFrontFace:Bool

Specifies whether or not front or back-facing polygons can be culled.

culling:Bool

Activates culling of polygons.

depthTest:Bool

Activates depth comparisons and updates to the depth buffer.

offsets:Bool

Activates adding an offset to depth values of polygon's fragments

polygonOffset:Float

The polygon offset. Setting this property to anything other than 0 will automatically enable polygon offset fill.
