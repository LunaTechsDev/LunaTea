@:native("PIXI.Attribute")

Available on all platforms

Constructor

new(buffer:String, ?size:Int, ?normalized:Bool, ?type:Float, stride:Float, start:Int)

Holds the information for a single attribute structure required to render geometry.

This does not contain the actual data, but instead has a buffer id that maps to a PIXI.Buffer This can include anything from positions, uvs, normals, colors etc.

Parameters:

buffer

{String} the id of the buffer that this attribute will look for

size

{Int} the size of the attribute. If you have 2 floats per vertex (eg position x and y) this would be 2.

normalized

{Bool} should the data be normalized.

type

{Float} what type of number is the attribute. Check {PIXI.TYPES} to see the ones available

stride

{Float} How far apart (in floats) the start of each value is. (used for interleaving data)

start

{Int} How far into the array to start reading values (used for interleaving data)
