Constructor

@:value({ nativeLines : false })new(nativeLines:Bool = false)

The Graphics class contains methods used to draw primitive shapes such as lines, circles and rectangles to the display, and color and fill them.

@class @extends Container @namespace PIXI

Parameters:

{Bool}

[nativeLines=false] - If true the lines will be draw using LINES instead of TRIANGLE_STRIP
